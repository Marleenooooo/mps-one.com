# UI Theme & I18N

Palet neon korporat dengan mode gelap/terang, target aksesibilitas, dan micro‑interactions.

## Dasar Tema
- Mode gelap: midnight blue, warna neon modul.
- Mode terang: putih/biru‑abu, aksen profesional.

## Interaksi
- Hover tombol: scale(1.02), glow, box‑shadow.
- Hover kartu: translateY(-2px), bayangan meningkat.

## I18N
- Bahasa: English (`en`), Indonesia (`id`).
- Preferensi disimpan di localStorage; deteksi OS; toggle manual.
