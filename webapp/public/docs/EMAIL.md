# Email & Communication Hub

## Purpose
- Centralize supplier‑client communication with in‑app threads and email mirroring.

## Features
- Templates with CC/BCC rules per client type.
- Read receipts, @mentions, and file sharing from Document Manager.
- Threads linked to PR/Quote/PO/Delivery/Invoice contexts.

## Status
- Designed for integration; in front‑end mode, email can be simulated until backend is connected.
